# Midterm Project – Cybersecurity Plan for AI-Integrated IIoT System

## Overview
This project focused on designing a cybersecurity plan for a smart factory that uses AI and Industrial IoT (IIoT) systems.

## Problem
Smart factories improve efficiency but introduce new security risks due to connected devices, networks, and AI systems.

## Approach
- Identified vulnerabilities across:
  - Devices
  - Networks
  - Data
  - Applications
  - AI systems
  - Human users
- Designed a layered cybersecurity defense strategy
- Simulated attacks to test system security

## Key Concepts Learned
- Security must be applied at every layer of the system
- AI systems have unique vulnerabilities (data manipulation, adversarial inputs)
- Human error is one of the biggest risks
- Network segmentation and encryption are critical

## Results / Findings
- MFA successfully prevented unauthorized access
- Encryption protected data from interception
- Network segmentation limited attack spread
- Some phishing vulnerabilities remained

## Improvements
- Better employee training
- Stronger AI protections
- Continuous monitoring and testing

## Simple Reflection
This project showed me how complex cybersecurity is in real systems. It’s not just about protecting one part, but everything working together. It also made me realize that human behavior is one of the biggest security risks.
