# Lab 2 – TensorFlow Lite Model Conversion

## Overview
This lab focused on converting a machine learning model into TensorFlow Lite format for deployment on edge devices.

## Objectives
- Understand TensorFlow Lite
- Convert a TensorFlow model into a .tflite model
- Optimize the model for performance
- Test inference after conversion

## What We Did
- Loaded and verified a trained TensorFlow model
- Converted it into TensorFlow Lite format
- Applied optimization techniques to reduce size
- Tested the model using the TensorFlow Lite interpreter

## Key Concepts Learned
- Deployment is just as important as model training
- TensorFlow Lite allows models to run on small devices
- Optimization improves efficiency without breaking functionality
- Edge AI reduces latency and improves privacy

## Challenges
- Understanding differences between TensorFlow and TensorFlow Lite
- Interpreting conversion outputs and warnings
- Ensuring the model still worked after conversion

## Simple Reflection
This lab showed me that building a model is only part of the process. I learned that deployment and efficiency are just as important, especially for real-world applications. It also helped me understand how AI can run on smaller devices like phones or IoT systems.
