# Lab 6 – IIoT Time Series Forecasting

## Overview
This lab focused on forecasting temperature data collected from IoT sensors using time series analysis and AutoML techniques.

## Objectives
- Clean and preprocess IoT data
- Build forecasting models
- Evaluate model performance
- Explore generative AI for data augmentation

## What We Did
- Cleaned and prepared temperature dataset
- Extracted time-based features (day, month, hour)
- Resampled data into daily averages
- Used AutoML (StatsForecast) to train models
- Evaluated models like MSTL and AutoARIMA
- Built a Variational Autoencoder (VAE) to generate synthetic data

## Key Concepts Learned
- Time series data requires special preprocessing
- Feature engineering improves model performance
- AutoML simplifies model selection
- Synthetic data can improve model robustness

## Challenges
- Handling noisy and irregular IoT data
- Understanding time series forecasting methods
- Working with AutoML tools

## Simple Reflection
This lab helped me understand how real-world IoT data is messy and needs a lot of preprocessing. I also learned that choosing the right features can make a big difference in predictions. The idea of using AI to generate more data was really interesting and showed how models can be improved beyond just collecting more data.
