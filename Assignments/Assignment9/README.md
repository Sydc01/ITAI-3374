# Assignment – Autonomous Agents in Industry 4.0

## Overview
This assignment explored how autonomous agents are used in modern industrial systems, particularly in smart factories.

## Problem
Traditional systems follow fixed instructions and struggle in dynamic environments.

## Approach
- Studied reinforcement learning models
- Analyzed a warehouse simulation using DDQN
- Evaluated how agents learn through rewards and penalties

## Key Findings
- Agents can learn efficient behavior over time
- Systems become more flexible and adaptive
- Decentralized decision-making improves reliability

## Limitations
- Tested only in simulation
- Real-world environments are more complex
- Training instability can occur

## Simple Reflection
This assignment helped me understand how AI systems can learn on their own through experience. It also showed me that even if something works in a simulation, it doesn’t always mean it will work perfectly in real life.
