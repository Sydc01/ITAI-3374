# Assignment 6 – Autonomous Agents in Industry 4.0

## Overview
This assignment focused on how autonomous agents are used in Industry 4.0 systems, particularly in smart factories and automated environments.

## Problem
Traditional systems rely on fixed instructions and struggle to adapt to changing environments. The challenge is to create systems that can make decisions independently and respond dynamically.

## Approach
- Studied reinforcement learning techniques
- Analyzed a warehouse simulation using multiple agents
- Evaluated how agents learn using rewards and penalties
- Reviewed the use of Double Deep Q-Networks (DDQN)

## Key Concepts Learned
- Autonomous agents can learn through trial and error
- Reinforcement learning allows systems to adapt over time
- Decentralized systems improve reliability
- Energy management is important in real-world systems

## Results / Findings
The agents were able to:
- Complete tasks efficiently
- Avoid collisions
- Manage battery usage
- Adapt to the environment without fixed instructions

## Limitations
- The system was tested in simulation only
- Real-world environments are more complex
- Training can be unstable at times

## Simple Reflection
This assignment helped me understand how AI systems can learn on their own instead of being programmed step-by-step. It also made me realize that even if something works perfectly in a simulation, real-world situations are much more unpredictable.
